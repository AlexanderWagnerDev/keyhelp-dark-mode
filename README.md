# KeyHelp Dark Mode – Neutral Gray Theme

This repository provides a custom **dark mode CSS theme** for the KeyHelp control panel.  
The focus is on a neutral gray look (no strong blue tint), good contrast and readability, including code editors and disabled controls.

## Features

- Neutral dark background for the whole panel  
- Styled header, sidebar, cards and panels  
- Unified table design for system information and lists  
- Dark form inputs, selects and textareas  
- Proper styling for disabled buttons and inputs  
- Customized colors for the built‑in Ace code editor  
- Improved tabs, breadcrumbs, badges and tooltips  

## Installation

1. Log in to your **KeyHelp admin panel**.  
2. Go to: `Settings → Appearance → White Label → Edit (All languages)`  
3. Open the **“Additional CSS”** textarea.  
4. Copy the contents of `style.css` from this repository and paste it into that field.  
5. Save the changes and reload the panel in your browser.  

If the theme does not show up immediately, clear the template cache (via the toolbox, if available) and perform a hard reload in your browser.

## Files

- `style.css` – main stylesheet containing all dark mode adjustments for KeyHelp.

## Customization

You can easily adapt the theme:

- Change the base background color in the `html, body` rule.  
- Adjust the link accent colors in the `a` and `a:hover` rules.  
- Tweak card and table colors in the `.card`, `.panel` and `.table` sections.  
- Modify code syntax colors in the `Ace editor` section.

# KeyHelp Dark Mode – Neutrales Grau Theme

Dieses Repository stellt ein benutzerdefiniertes **Dark‑Mode‑CSS‑Theme** für das KeyHelp‑Control‑Panel bereit.  
Der Schwerpunkt liegt auf einem neutralen Grauton (kein starker Blaustich), gutem Kontrast und guter Lesbarkeit – auch im Code‑Editor und bei deaktivierten Elementen.

## Funktionen

- Neutrales dunkles Hintergrunddesign für das gesamte Panel  
- Angepasster Header, Sidebar, Cards und Panels  
- Einheitliches Tabellendesign für Systeminformationen und Listen  
- Dunkle Formularfelder, Selects und Textareas  
- Angepasste Darstellung von deaktivierten Buttons und Eingabefeldern  
- Eigene Farben für den integrierten Ace‑Code‑Editor  
- Überarbeitete Tabs, Breadcrumbs, Badges und Tooltips  

## Installation

1. Im **KeyHelp‑Adminbereich** anmelden.  
2. Navigieren zu: `Einstellungen → Erscheinungsbild → White Label → Bearbeiten (Alle Sprachen)`  
3. Das Textfeld **„Zusätzliches CSS“** öffnen.  
4. Den Inhalt der Datei `style.css` aus diesem Repository in dieses Feld einfügen.  
5. Änderungen speichern und das Panel im Browser neu laden.  

Falls die Änderungen nicht sofort sichtbar sind, ggf. den Template‑Cache leeren (über die Toolbox, falls vorhanden) und einen Hard‑Reload im Browser durchführen.

## Dateien

- `style.css` – Hauptstylesheet mit allen Dark‑Mode‑Anpassungen für KeyHelp.

## Anpassung

Das Theme lässt sich leicht weiter anpassen:

- Basis‑Hintergrundfarbe in der Regel `html, body` ändern.  
- Link‑Akzentfarben in den Regeln `a` und `a:hover` anpassen.  
- Farben für Cards und Tabellen in den Abschnitten `.card`, `.panel` und `.table` tweaken.  
- Syntax‑Farben im Abschnitt **Ace‑Editor** nach eigenen Vorlieben ändern.
